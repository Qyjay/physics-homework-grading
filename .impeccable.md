# Design Context

## Users
- **Teachers**: Upload standard answers, view grading results, manage classes
- **Students**: Submit homework, view grading feedback
- **Context**: Classroom education, high school physics

## Brand Personality
- **Voice**: 活力教育 (Vibrant Educational)
- **Tone**: Young, friendly, encouraging — like Duolingo or Quizlet
- **Emotional goals**: Make learning physics feel achievable and engaging

## Aesthetic Direction
- **Visual tone**: Vibrant but not childish — education with energy
- **Theme**: Both light and dark mode (user switchable)
- **Reference**: Duolingo's playfulness meets Linear's polish
- **Anti-reference**: No boring enterprise gray, no generic admin dashboards

## Design Principles
1. **Encouraging feedback** — correct answers feel rewarding, wrong answers feel constructive
2. **Progress visibility** — clear steps, progress bars, completion states
3. **Accessible contrast** — WCAG AA minimum, not just "looks good on my screen"
4. **Motion with purpose** — animations celebrate progress, not just decoration
5. **Typography hierarchy** — clear information density, not overwhelming

## Color Direction
- **Light mode**: Warm off-white background, vibrant accent colors
- **Dark mode**: Deep charcoal, not pure black
- **Accent**: Physics-themed (electric blue, scientific feel)
- **Feedback colors**: Success green, encouraging orange, not harsh red

## Motion Direction
- **Entrance**: Staggered fade-up reveals on page load
- **Feedback**: Scale + color shifts on correct/incorrect
- **Progress**: Smooth counter animations, celebratory micro-interactions
- **Transitions**: 300ms ease-out-quart as base timing

## Technical Constraints
- Vue 3 Composition API with `<script setup>`
- Ant Design Vue component library
- TypeScript throughout
- CSS variables for theming
