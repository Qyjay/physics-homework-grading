# 高中物理作业智能批改系统 — 项目章程

> 课程：软件项目管理 | 项目编号：第 16 题 | 团队规模：5 人
> 编制日期：2026-03-25 | 编制人：赵麒杰（PM）

## 一、项目背景

高中物理作业具有多模态特征（文本、公式、示意图），人工批改效率低、反馈延迟、标准不统一。

## 二、项目目标

开发基于 NLP + CV + LLM 的高中物理作业智能批改系统：
1. 图像输入处理（手机拍照/扫描件）
2. 多模态识别（文字/公式/图形分离）
3. 智能批改（基于物理知识库 + 大模型逐步检查）
4. 反馈生成（错误原因 + 改正建议）
5. 智能体协作（任务分解 + 分步批改）

### 成功标准

| 指标 | 目标值 |
|------|--------|
| 文字 OCR 准确率 | ≥ 90% |
| 公式识别准确率 | ≥ 85% |
| 受力图关键元素识别率 | ≥ 80% |
| 批改与人工一致率 | ≥ 80% |
| 单题平均批改时间 | ≤ 15 秒 |
| 系统可用性 | ≥ 95% |

## 三、系统架构（8 大模块）

1. **图像预处理** — OpenCV（去噪/校正/透视/光照）
2. **图像分割** — DocLayout-YOLO + 连通域分析
3. **OCR 文字识别** — PaddleOCR + 多模态大模型校验
4. **公式识别** — UniMERNet（主）+ Mathpix API（备选）
5. **画图分析** — YOLO 目标检测 + 多模态大模型（最大技术挑战 ⭐⭐⭐⭐⭐）
6. **多模态关联** — 文本+公式+图像 → 结构化答案表示
7. **RAG 知识库** — FAISS/Milvus + BGE Embedding + 混合检索
8. **大模型批改 + 智能体** — InternLM-Math API + 多 Agent 任务分解
