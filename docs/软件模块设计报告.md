## 一、项目概述

### 1.1 项目背景

高中物理作业智能批改系统是一个结合计算机视觉（CV）、光学字符识别（OCR）和大语言模型（LLM）的智能化教育平台。系统支持用户上传学生手写物理作业图片，自动进行图像预处理、区域分割、文字/公式识别，并与标准答案进行对比，最终给出分步评分、错误分析和改进建议。

#### 2.2.1 路由模块 (router/index.ts)

**职责**：管理前端路由配置，实现页面导航和路由守卫。

**核心功能**：
- 路由配置：定义所有页面路径和组件映射
- 路由守卫：保护需要认证的页面，未登录重定向到登录页
- 懒加载：所有页面组件采用懒加载，优化首屏加载速度

**关键代码**：
```typescript
// 路由配置
const routes: RouteRecordRaw[] = [
  {
    path: '/login',
    name: 'Login',
    component: () => import('@/views/Login.vue'),
    meta: { requiresAuth: false },
  },
  {
    path: '/',
    component: () => import('@/views/Layout.vue'),
    meta: { requiresAuth: true },
    children: [
      { path: '', name: 'Home', component: () => import('@/views/Home.vue') },
      { path: 'upload', name: 'Upload', component: () => import('@/views/Upload.vue') },
      { path: 'result/:gradingId', name: 'Result', component: () => import('@/views/Result.vue') },
      { path: 'answers', name: 'Answers', component: () => import('@/views/Answers.vue') },
      { path: 'classes', name: 'Classes', component: () => import('@/views/Classes.vue') },
      { path: 'history', name: 'History', component: () => import('@/views/History.vue') },
    ],
  },
]

// 路由守卫
router.beforeEach((to, _from, next) => {
  const authStore = useAuthStore()
  if (to.meta.requiresAuth && !authStore.isAuthenticated) {
    next({ name: 'Login', query: { redirect: to.fullPath } })
  } else {
    next()
  }
})
```

**接口设计**：
| 接口 | 说明 |
|------|------|
| `router.push(path)` | 编程式导航 |

#### 2.2.2 认证状态模块 (stores/auth.ts)

**职责**：管理用户认证状态，包括登录、注册、登出和Token管理。

**核心功能**：
- 用户登录：调用后端登录API，保存JWT Token
- 用户注册：调用后端注册API
- Token管理：本地存储Token，axios请求自动附加
- 状态持久化：刷新页面后恢复登录状态

**关键代码**：
```typescript
export const useAuthStore = defineStore('auth', () => {
  const token = ref<string | null>(localStorage.getItem('token'))
  const user = ref<User | null>(null)

  const isAuthenticated = computed(() => !!token.value)

  async function login(username: string, password: string) {
    // 调用登录API，保存token
    const response = await axios.post(`${API_BASE_URL}/auth/login`, formData, ...)
    token.value = access_token
    localStorage.setItem('token', access_token)
    axios.defaults.headers.common['Authorization'] = `Bearer ${access_token}`
  }

  function logout() {
    token.value = null
    user.value = null
    localStorage.removeItem('token')
    delete axios.defaults.headers.common['Authorization']
  }

  return { token, user, isAuthenticated, login, register, logout }
})
```

**状态结构**：
| 状态 | 类型 | 说明 |
|------|------|------|
| token | string \| null | JWT访问令牌 |
| user | User \| null | 当前用户信息 |
| isAuthenticated | boolean | 是否已认证（计算属性） |

**关键代码**：
```typescript
// 创建 axios 实例
const api = axios.create({
  baseURL: '/api/v1',
  timeout: 60000, // 批改可能需要较长时间
})

// 请求拦截器
api.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})

// 响应拦截器
api.interceptors.response.use(
  response => response.data,
  (error: AxiosError) => {
    const message = error.response?.data?.detail || error.message
    return Promise.reject(new Error(message))
  }
)

// API函数示例
export async function uploadImage(file: File): Promise<ApiResponse<ImageUploadResponse>> {
  const formData = new FormData()
  formData.append('file', file)
  return api.post('/images/upload', formData, {
    headers: { 'Content-Type': 'multipart/form-data' },
  })
}

export async function submitGrading(data: GradingSubmitRequest): Promise<ApiResponse<GradingSubmitResponse>> {
  return api.post('/grading/submit', data)
}
```

```typescript
// 批改进度状态
const stepStatus = ref<Array<'wait' | 'process' | 'finish' | 'error'>>(
  ['wait', 'wait', 'wait', 'wait', 'wait']
)
const steps = ['上传', '预处理', '识别', '批改', '完成']

// 开始批改
async function handleUpload() {
  uploading.value = true
  stepStatus.value = ['process', 'wait', 'wait', 'wait', 'wait']

  try {
    const uploadRes = await uploadImage(file)
    const imageId = uploadRes.data?.image_id
    stepStatus.value[0] = 'finish'

    const gradingRes = await submitGrading({
      image_id: imageId,
      standard_answer_id: standardAnswerId.value,
    })

    startStatusPolling(gradingRes.data.grading_id)
  } catch (error) {
    stepStatus.value = ['error', 'error', 'error', 'error', 'error']
  }
}

// 状态轮询
function startStatusPolling(grdId: string) {
  statusPollTimer = window.setInterval(async () => {
    const res = await getGradingStatus(grdId)
    if (res.data?.progress >= 100) {
      clearInterval(statusPollTimer)
      router.push(`/result/${grdId}`)
    }
  }, 1000)
}
```

**关键代码**：
```typescript
// 表单结构
const form = ref({
  question_title: '',
  subject: '物理',
  grade: '高一',
  chapter: '',
  steps: [{ order: 1, description: '', content: '', formula: '', score: 0 }],
  total_score: 10,
  tags: [],
})

// 添加步骤
function addStep() {
  const newOrder = form.value.steps.length + 1
  form.value.steps.push({ order: newOrder, description: '', content: '', formula: '', score: 0 })
  updateTotalScore()
}

// 更新总分
function updateTotalScore() {
  form.value.total_score = form.value.steps.reduce((sum, step) => sum + (step.score || 0), 0)
}

// 提交表单
async function handleSubmit() {
  const submitData = {
    ...form.value,
    steps: form.value.steps.filter(s => s.description && s.content),
  }
  if (isEdit.value) {
    await updateAnswer(currentAnswerId.value, submitData)
  } else {
    await createAnswer(submitData)
  }
}
```
