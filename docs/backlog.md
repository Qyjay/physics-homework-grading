## 一、项目概述

**目标**：开发基于 NLP + CV + LLM 的高中物理作业智能批改系统

**技术栈**：
- 前端：Vue 3 + TypeScript + Vite + Pinia + Vue Router
- 后端：FastAPI + Python
- 图像处理：OpenCV + PaddleOCR + DocLayout-YOLO
- 公式识别：UniMERNet
- 知识库：FAISS + BGE Embedding
- 大模型：InternLM-Math API

**团队分工**：
| 角色 | 姓名 | 职责 |
|------|------|------|
| PM + 前端 + 基础设施 | 赵麒杰 | 项目管理、前端开发、基础设施（Sprint 0已完成） |
| CV开发 | 武英文 | 图像预处理、分割、OCR |
| AI开发1 | 李传宇 | 公式识别、多模态关联 |
| AI开发2 | 王硕 | RAG知识库、大模型集成、后端API |
| 测试/文档 | 代树衡 | 测试、文档、竞品调研 |
